1. JVM 에 대하여 설명하시오.(인터넷에서 찾아 보고 한번 정도는 읽어 보세요)
	java 애플리케이션을 클래스로더를 통해 읽어들여서 API와 함께 실행하는 것이다.
	또한 java와 os 사이에서 중개자역할을 하여 os에 구애받지 않고 재사용을 가능하게 해준다.

2. 일반적인 프로그램과 java 가 실행되는 차이점은(오늘 배운 PPT 그림 참고 하세요).
	일반적인 프로그램은 특정 os에 맞춰져서 만들어지기에 os가 맞지 않으면 실행이 되지 않지만
	java의 경우 JVM을 통하여 os에 구애받지 않고 실행할 수 있다는 차이점이 있다.
	따라서 일반적인 프로그램은 프로그램-실행-하드웨어(os) 순으로 실행이 되지만
	java는 프로그램-JVM-실행-하드웨어(os)의 순으로, 한 단계를 더 거쳐서 실행이 된다.

-프로그램밍 문제 
국어 90 영어 70 수학 60 점의 평균 및 총합을 출력하시오.
변수명 유의해서 프로그램 짜주세요.

public class Average {
	public static void main(String[] args) {
		int kor = 90;
		int eng = 70;
		int mat = 60;

		int total = kor + eng + mat;
		double avg = (double) total / 3;
		System.out.println(avg);
	}
}
